## Commands

Slash commands are in `commands/`. Each `.md` file is a command (e.g. `/frontend-design`). Read the file when the user invokes the command.

## Project-Local Overrides

If the project has its own `.claude/CLAUDE.md` entries or `.claude/commands/` files that conflict with these shared rules, prefer the project-local ones.
